Hello There

Il s'agit d'un site web simple permettez de vérifier si la machine à laver est libre. Vous n'aurez pas besoin de vous promener avec vos vêtements ou de demander dans le groupe whatsapp si tout le monde utilise ce site.

Instructions :

  1. Entrez votre nom à la fin de ce texte et cliquez sur ok.
  2. Permettre au site web de vous envoyer des notifications
  3. Si la machine à laver est libre, vous pourrez programmer une alarme, afin de pouvoir récupérer vos vêtements à temps.
  4. Si le lave-linge est occupé, vous pourrez voir qui l'utilise et combien de temps il reste.
  5. Une fois que vous avez terminé avec vos vêtements, n'oubliez pas de basculer le bouton sur Libre.

Merci
